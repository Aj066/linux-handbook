# Level 3 — Advanced (Production-Ready Systems)

## 1. systemd Mastery
- Create custom services  
- Manage dependencies  
- Handle restarts and failures  
- Understand systemd journaling  

## 2. Security & SSH Hardening
- Disable root login  
- Enforce key-based authentication  
- Disable unnecessary SSH functions  
- Add fail2ban or similar protections  

## 3. LVM Storage Management
- PV → VG → LV lifecycle  
- Mounting and resizing volumes  
- Online filesystem expansion  

## 4. Firewall Configuration
- UFW (Ubuntu) rules  
- firewalld (RHEL) rules  
- Principle of least privilege  

## 5. Log Rotation Strategy
- Log retention policies  
- logrotate rules  
- Prevent disk full incidents
